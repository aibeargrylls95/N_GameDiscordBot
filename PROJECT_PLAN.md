# 🎮 N_GameDiscordBot 프로젝트 계획서 (Project Plan)

이 문서는 **N_GameDiscordBot**의 전체 파일 구조, 각 파일의 역할 및 향후 확장 계획(LOL 패치노트 & 메이플 URL 파라미터 연동)을 설명하는 아키텍처 가이드입니다.

---

## 📂 1. 프로젝트 폴더 구조 (Directory Structure)

현재 프로젝트 폴더 구조는 1인 개발 목적에 맞게 군더더기 없이 슬림하고 직관적으로 설계되었습니다.

```text
N_GameDiscordBot/
├── .env                  # [보안] 봇 토큰 등 핵심 민감정보 저장 (GitHub 절대 금지)
├── .gitignore            # [보안] GitHub 업로드에서 제외할 파일 목록 정의
├── package.json          # [환경] Node.js 프로젝트 설정 및 설치할 라이브러리 목록
├── index.js              # [메인] 디스코드 봇의 메인 구동 로직 (봇의 뇌)
└── PROJECT_PLAN.md       # [문서] 본 프로젝트 설계 및 개발 계획서 (현재 파일)
```

---

## 🔍 2. 각 파일의 역할 및 설명 (File Responsibilities)

### 🛡️ 1) `.gitignore`
* **역할**: GitHub에 코드를 올릴 때, **절대 올라가면 안 되는 파일**을 미리 설정하여 보안 사고를 예방합니다.
* **제외 대상**: 
  * `.env`: 디스코드 봇 토큰이 포함되어 있어 유출 시 타인이 봇을 해킹할 수 있습니다.
  * `node_modules/`: 언제든 다시 다운로드할 수 있는 대용량 외부 라이브러리 폴더입니다.

### ⚙️ 2) `package.json`
* **역할**: 이 프로젝트가 사용하는 Node.js 환경의 설정서입니다.
* **핵심 설정**:
  * `"type": "module"`: 최신 자바스크립트 표준 문법(ES Module)을 지원합니다.
  * `"dependencies"`: 디스코드 연동을 위한 `discord.js`와 환경변수를 다루기 위한 `dotenv`를 설치합니다.

### 🔑 3) `.env`
* **역할**: 봇 토큰 등 핵심 비밀키를 로컬 컴퓨터 안에서 안전하게 관리합니다. (이 파일은 미니 PC 이전 시 직접 복사하여 옮깁니다.)

### 🧠 4) `index.js` (봇의 심장이자 뇌)
* **역할**: 디스코드 서버와 실시간으로 통신하며, 이벤트(메시지 수신 등)를 감지하고 매크로 로직을 처리합니다.
* **현재 구현 기능**:
  * **온라인 접속**: 실행 시 디스코드 서버에 봇이 온라인으로 켜집니다. (오직 `#봇-응답` 채널 전용)
  * **봇 정보 안내**: `봇 정보`, `!봇정보`, `!정보` 입력 시 사용 가능한 명령어 안내 카드 반환.
  * **실시간 롤 패치 조회**: `!패치`, `!롤패치` 등 입력 시 공식 홈페이지 실시간 크롤링 및 파싱 링크 반환.
  * **실시간 이전 롤 패치 조회**: `!지난패치`, `!이전패치` 입력 시 직전 회차 패치노트 링크 반환.
  * **롤 통계 바로가기**: `!ps`, `!opgg` 대소문자 무관 바로가기 카드 반환.

## 🔮 3. 향후 확장 계획: LOL 패치노트 & 메이플 URL 매크로

이 봇은 게임 정보 제공을 극대화하는 **게임 유틸리티 봇**으로 성장합니다. 다음 기능이 순차적으로 추가됩니다.

```text
N_GameDiscordBot/
├── ... (기존 파일)
└── commands/             # [향후 추가] 기능별 분리 폴더
    ├── lol.js            # LOL 관련 기능 (패치노트 조회 등)
    └── maple.js          # 메이플 관련 기능 (URL 생성 등)
```

### 🏆 1) 롤(LOL) 관련 기능 고도화
* **기능**: 단순한 패치노트 조회 외에 챔피언명 입력 시 특정 사이트 티어 연동 등 추가 가능.

### 🍁 2) 메이플스토리 URL 파라미터 매크로 기능
* **기능**: 사용자가 캐릭터 닉네임을 입력하면 자주 쓰이는 유용한 사이트들의 주소에 파라미터를 결합하여 바로 갈 수 있는 주소를 원클릭 링크로 만들어 줍니다.
* **연동 사이트 예시**: 
  * Maple.gg (전투력/랭킹 조회): `https://maple.gg/u/[캐릭터명]`
  * 메이플 공식 홈페이지 캐릭터 조회 등
* **사용 예시**: `!메이플 [캐릭터명]` 입력 시 봇이 해당 유저의 랭킹 및 세부 스펙을 바로 확인할 수 있는 커스텀 링크 메시지 카드 출력.
